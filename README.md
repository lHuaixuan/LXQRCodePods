# LXQRCodePods
It's a simple demo about QRCode. 可以扫描二维码、以及识别图片中的二维码



第一步：导入<br>
1.1 导入CocoaPods<br>
    ```pod "LXQRCodePods"```<br>

1.2 项目添加权限 info.plist 中<br>
    ```Privacy - Camera Usage Description<br>
    Privacy - Photo Library Usage Description```<br>

第二步：使用<br>
2.1 导入头文件<br>
    ```#import "LXQRCode_VC.h"```<br>
